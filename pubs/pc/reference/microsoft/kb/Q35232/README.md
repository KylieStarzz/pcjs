	Article: Q35232
	Product: Microsoft C
	Version(s): 6.x 1.00
	Operating System: MS-DOS
	Flags: ENDUSER |
	Last Modified: 19-SEP-1988
	
	CPANEL will not work in the OS/2 DOS 3.x box unless you load the DOS
	mouse driver rather than the OS/2 mouse driver.
