## Q31280: Purpose of ECH.EXE with the Microsoft Editor

	Article: Q31280
	Version(s): 1.00
	Operating System: OS/2
	Flags: ENDUSER |
	Last Modified: 8-JUN-1988
	
	ECH.EXE is used by the editor to generate the list of files you
	receive when you do <arg> "*.C" <setfile>.
